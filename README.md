# map_under
echarts结合vue 实现 地图下钻
使用element-ui 作为错误信息弹出框
目前只有成都市json信息齐全，后期跟新其他城市信息
也可以自己获取去地图选择器("地址":"http://datav.aliyun.com/tools/atlas/#&lat=31.769817845138945&lng=104.29901249999999&zoom=4")      
下载json数据，放入json中

## 项目启动
```
npm install
```
```
npm run serve
```

### 项目打包
```
npm run build
```
